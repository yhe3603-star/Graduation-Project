# 页面视图目录说明

## 文件夹结构

本目录包含项目中所有的页面组件，每个文件对应一个完整的页面。

```
views/
├── About.vue           # 关于我们
├── Admin.vue           # 管理后台
├── Feedback.vue        # 意见反馈
├── GlobalSearch.vue    # 全局搜索
├── Home.vue            # 首页
├── Inheritors.vue      # 传承人
├── Interact.vue        # 互动中心
├── Knowledge.vue       # 知识库
├── NotFound.vue        # 404页面
├── PersonalCenter.vue  # 个人中心
├── Plants.vue          # 药材
├── Qa.vue              # 问答
├── Resources.vue       # 资源
└── Visual.vue          # 可视化
```

## 页面开发规范

1. **命名规范**：页面文件使用大驼峰命名法
2. **组件结构**：每个页面组件包含template、script、style三部分
3. **路由配置**：页面路由在 `router/index.js` 中配置
4. **数据获取**：使用 `utils/request.js` 进行API请求
5. **状态管理**：使用Pinia进行状态管理
6. **样式规范**：使用CSS变量和统一的样式规范

## 路由配置

所有页面的路由配置在 `router/index.js` 文件中，包括：
- 路由路径
- 组件导入
- 路由元信息（标题、权限等）
- 嵌套路由（如管理后台）

## 页面加载流程

1. 用户访问URL
2. 路由匹配到对应的页面组件
3. 页面组件加载，执行 `onMounted` 钩子
4. 调用API获取数据
5. 数据加载完成后渲染页面
6. 用户与页面交互，触发相应的事件处理
