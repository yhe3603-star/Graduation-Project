## 一、项目概述

本项目是一个基于 Vue 3 + Spring Boot 的侗族医药文化遗产数字化展示平台，旨在解决"非遗资源散落、传承渠道狭窄"的痛点，为侗乡医药非遗提供可持续的数字化传播与保护平台。

### 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端 | Vue 3 + Vite + Element Plus + Pinia + ECharts | Vue 3.4+ |
| 后端 | Spring Boot + MyBatis Plus + Spring Security + JWT | Spring Boot 3.1.12 |
| 数据库 | MySQL 8.0 + Redis 7.0 | - |
| 部署 | Docker + Nginx + GitHub Actions | - |

## 五、解决办法

### 1. 答题题目补充方案

**需要补充**: 21道题目（11道基础题 + 10道进阶题）

**建议题目内容**:

#### 基础题（11道）
```sql
-- 侗族医药基础知识
1. 侗族医药是哪个民族的传统医学？（侗族）
2. 侗族药浴的主要功效是什么？（祛风除湿、舒筋活络）
3. 钩藤主要用于治疗什么疾病？（惊风、头痛）
4. 侗医艾灸的主要材料是什么？（艾叶）
5. 透骨草的主要功效是什么？（祛风除湿、活血止痛）
6. 侗族医药的"三位一体"是指什么？（天、地、人）
7. 八角莲主要用于治疗什么？（蛇伤）
8. 侗医推拿的主要作用是什么？（舒筋活络）
9. 金银花的主要功效是什么？（清热解毒）
10. 侗族药浴的水温一般是多少？（40-45℃）
11. 杜仲的主要功效是什么？（补肝肾、强筋骨）
```

#### 进阶题（10道）
```sql
-- 侗族医药进阶知识
1. 侗族医药的"天、地、气、水、人"五位一体理论是什么？
2. 侗医治疗蛇伤的三步法是什么？
3. 侗族药浴配方中的"君臣佐使"如何配伍？
4. 侗医火针疗法的适应症有哪些？
5. 侗族唐氏武医推拿术的特点是什么？
6. 侗药命名的三大方法是什么？
7. 侗医治疗风湿痹痛的经典方剂组成？
8. 侗族产后调理方的用药原理？
9. 侗医小儿推拿的常用穴位有哪些？
10. 侗药炮制的主要方法有哪些？
```
